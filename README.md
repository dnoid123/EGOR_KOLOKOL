# EGOR_KOLOKOL

Коликов Егор

Задачник — это простое desktop-приложение с графическим интерфейсом на Python. Программа помогает случайным образом выбирать задачу из предопределённого списка и ведёт историю всех сгенерированных задач.

Основные возможности:
- Генерация случайной задачи (учёба, спорт, работа)
- Отображение текущей задачи и её типа
- История всех сгенерированных задач
- Фильтрация истории по типу задачи
- Сохранение истории в JSON-файл

Запуск программы с помощью файла `main.py`

Пример 1:
Ситуация: Вы хотите начать день с полезных дел, но не можете выбрать, с чего начать.

Действия:
1. Запустите программу
2. Нажмите "Сгенерировать задачу" 3 раза подряд
Результат: 
У вас готов план из 3 задач на утро!

Пример 2: 
Ситуация: Вы планируете дела на неделю (пн-пт = 5 дней, по 2 задачи в день = 10 задач)

Действия:

1. Нажмите "Сгенерировать задачу" 10 раз

2. Запишите полученные задачи по дням

3. Используйте фильтр, чтобы проверить баланс

Пример распределения:

Понедельник:
- 09:00 - Прочитать статью (учёба)
- 15:00 - Сделать зарядку (спорт)

Вторник:
- 10:00 - Написать отчёт (работа)
- 16:00 - Выучить 10 слов (учёба)

Среда:
- 09:30 - Отжаться 20 раз (спорт)
- 14:00 - Ответить на письма (работа)

Четверг:
- 11:00 - Посмотреть лекцию (учёба)
- 17:00 - Пробежать 1 км (спорт)

Пятница:
- 10:00 - Создать презентацию (работа)
- 15:30 - Решить 5 задач (учёба)
